## What You'll Learn

This module covers the foundational Linux skills every infrastructure engineer needs. You'll work in real Linux environments, completing hands-on exercises that teach practical skills.

### Module Objectives

- Navigate the Linux file system confidently
- Manage files and directories with command-line tools
- Understand and control Linux processes
- Create and manage systemd services
- Configure basic system settings

### Prerequisites

None! This module is designed for beginners. All you need is:
- A computer with Docker installed (for local sandbox) OR
- An AWS account (for cloud sandbox)
- The `infra-learn` CLI installed

### How It Works

1. **Read the lesson** - Each exercise starts with explanatory content
2. **Launch the sandbox** - Use the CLI to spin up your practice environment
3. **Complete the task** - Work through the exercise in your sandbox
4. **Verify your work** - Run verification to check your solution
5. **Move to the next exercise** - Your progress is tracked automatically
